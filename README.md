# Front маршруттар(бағыттар)

## Front-end-дегі URL-дар

### Сертификаттау(Auth)

| Атауы   |     URL     |   auth    | permission  |
| ------- | :---------: | :-------: | ----------- |
| кіру    |  `/login`   | `[false]` | `['guest']` |
| тіркелу | `/register` | `[false]` | `['guest']` |

### Негізгі интерфейс

| Атауы                 | URL |   auth   | permission |
| --------------------- | :-: | :------: | ---------- |
| үйренуші негізгі беті | `/` | `[true]` | `['ST']`   |
